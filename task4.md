# task_4_verilog_simulation



### Installing iverilog and gtkwave

- **For Ubuntu**

 Open the terminal and type the following to install iverilog and GTKWave
 ```
 $   sudo apt get update
 $   sudo apt get install iverilog gtkwave
 ```
